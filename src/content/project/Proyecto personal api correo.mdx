Este proyecto implementa una <ExternalLink href='https://aws.amazon.com/es/what-is/api/' text='API' /> desarrollada con <ExternalLink href='https://quarkus.io/' text='Quarkus' />, 
basada en la <ExternalLink href='https://blog.hubspot.es/website/que-es-arquitectura-hexagonal' text='arquitectura hexagonal' />, 
también conocida como arquitectura de puertos y adaptadores. Esta arquitectura se centra en el dominio, lo que la hace ideal para tecnologías o frameworks que aplican el diseño guiado 
por el dominio <ExternalLink href='https://blog.hubspot.es/website/que-es-ddd' text='(DDD, Domain-Driven Design)' />. 
La API está diseñada para gestionar formularios de contacto, siendo una solución adecuada para integrarse en diferentes aplicaciones web. Para más detalles sobre el desarrollo, 
visite el repositorio de GitHub.
