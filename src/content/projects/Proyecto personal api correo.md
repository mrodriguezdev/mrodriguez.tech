Este proyecto implementa una API desarrollada con <a target="_blank" href="https://quarkus.io/" style="text-decoration: none; color: var(--color-verde-lima);" onmouseover="this.style.textDecoration='underline';" onmouseout="this.style.textDecoration='none';">Quarkus</a>, basada en la <a target="_blank" href="https://blog.hubspot.es/website/que-es-arquitectura-hexagonal" style="text-decoration: none; color: var(--color-verde-lima);" onmouseover="this.style.textDecoration='underline';" onmouseout="this.style.textDecoration='none';">arquitectura hexagonal</a>, también conocida como arquitectura de puertos y adaptadores. Esta arquitectura se centra en el dominio, lo que la hace ideal para tecnologías o frameworks que aplican el diseño guiado por el dominio <a target="_blank" href="https://blog.hubspot.es/website/que-es-ddd" style="text-decoration: none; color: var(--color-verde-lima);" onmouseover="this.style.textDecoration='underline';" onmouseout="this.style.textDecoration='none';">(DDD, Domain-Driven Design)</a>. La API está diseñada para gestionar formularios de contacto, siendo una solución adecuada para integrarse en diferentes aplicaciones web. Para más detalles sobre el desarrollo, visite el repositorio de GitHub.
