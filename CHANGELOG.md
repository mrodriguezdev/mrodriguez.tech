# Changelog

Todos los cambios importantes de este proyecto se documentarán en este archivo.

## [2.0.1] - 2026-01-18
### Añadido
- Navegación de artículos anterior y siguiente dentro de cada post.

### Cambiado
- Refactor del listado de publicaciones usando un componente reutilizable (`PostList`).
- Ajustes de estilo y alineación en la navegación de artículos.

## [2.0.0] - 2026-01-14
### Cambiado
- Migración completa de estilos desde CSS puro a Tailwind CSS.
- Refactor de componentes para usar utilidades de Tailwind.
- Artículos adaptados al nuevo sistema de estilos.

## [1.0.0] - 2026-01-12
### Añadido
- Primera versión estable del blog con Astro.
- Estilos implementados con CSS puro.
- Contenido en Markdown.
- Despliegue en Vercel.
